  ## Installation

  First, clone the github repository:
  > git clone github.com/kjlomax/challenge-7

  Then install dependencies:
  > npm i

  ## Usage

  When you answer a prompt with the following input:
  > Title

  it will add your answer to the README file.
  > # Title
